# brawler_fighting_game
A 2D Street Fighter-style fighting game built in Python using Pygame, showcasing Object-Oriented Programming (OOP) concepts.

## 📌 OOP Project
This repository contains the source code for my final project. The project is heavily based on the excellent tutorial by **Coding With Russ**. 

* **Original Tutorial Link:** [Street Fighter Style Fighting Game in Python using Pygame](https://www.youtube.com/watch?v=s5bd9KMSSW4)
* **Author:** Coding With Russ
